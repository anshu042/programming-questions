## üìñ About

This repository contains a collection of Python scripts designed to solve common programming problems and algorithmic challenges. It serves as a resource for understanding fundamental concepts such as recursion, string manipulation, and mathematical logic.

## üìÇ Scripts Overview

Here is a detailed list of the programs included in this repository.

### Mathematical Operations

| Script | Description |
| :--- | :--- |
| **[armstrong-number.py](./armstrong-number.py)** | Determines if a number is an Armstrong number (where the sum of its digits raised to the power of the number of digits equals the number itself). |
| **[factorial.py](./factorial.py)** | Calculates the factorial of a non-negative integer using an iterative loop approach. It handles edge cases like negative numbers and zero. |
| **[fibonacci.py](./fibonacci.py)** | Generates a Fibonacci series up to `n` terms using a **recursive function**. It prompts the user for the number of terms to display. |
| **[prime-number.py](./prime-number.py)** | Checks if a user-inputted integer is a prime number. Uses a loop to test divisibility from 2 up to `n`. |
| **[even-odd.py](./even-odd.py)** | A simple utility to check if an input number is even or odd using the modulo operator. |

### String Manipulation

| Script | Description |
| :--- | :--- |
| **[string-palindrome.py](./string-palindrome.py)** | Checks if a given string is a palindrome (reads the same forwards and backwards) by manually reversing the string loop-by-loop. |
| **[count-vowels.py](./count-vowels.py)** | Iterates through a user-provided string and counts the total number of vowels (`a, e, i, o, u`), supporting both uppercase and lowercase characters. |
| **[reverse-string.py](./reverse-string.py)** | Reverses a given string or integer input using Python's list slicing syntax `[::-1]` for efficiency. |

### Array Logic

| Script | Description |
| :--- | :--- |
| **[min-max.py](./min-max.py)** | Finds the maximum and minimum values in a user-defined array (list) without using the built-in `min()` or `max()` functions, utilizing a linear scan instead. |
